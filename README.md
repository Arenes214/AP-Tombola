Client for AP Tombola, a classic italian bingo game implemented as a meta game for the Archipelago Multiworld Randomizer.

For information on how to play the game read the [APWorld Documentation](https://github.com/Arenes214/APWorld-Tombola/blob/master/docs/en_AP%20Tombola.md)

The client is also available as a [web page on itch.io](https://arenes214.itch.io/ap-tombola)
